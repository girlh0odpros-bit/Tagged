# Tagged 🏷️

## Étape 1 — Créer les tables dans Supabase
1. Va sur ton projet Supabase → **SQL Editor**
2. Ouvre le fichier `supabase-schema.sql` de ce dossier, copie tout son contenu
3. Colle-le dans le SQL Editor et clique **Run**

## Étape 2 — Lancer l'app en local
Il te faut [Node.js](https://nodejs.org) installé sur ton ordi (version 18+).

Dans ton terminal, à l'intérieur de ce dossier :

```
npm install
npm run dev
```

Ça va afficher une adresse du type `http://localhost:5173` — ouvre-la dans ton navigateur.

## Étape 3 — Créer ton compte
Sur l'écran de connexion, clique "Créer un compte", mets ton email + un mot de passe.
Supabase va t'envoyer un email de confirmation — clique dessus, puis reviens te connecter.

## Étape 4 — Utiliser l'app
1. Crée ta première collection (bouton "+ Nouvelle")
2. Ajoute tes premiers articles avec le bouton rose "+"

## Prochaine étape : héberger en ligne
Une fois que tout marche bien en local, on pourra déployer sur Vercel pour avoir une vraie URL accessible depuis ton téléphone.
